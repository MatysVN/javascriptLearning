Funkcje

Wprowadzenie



deklaracja funkcji:


function nazwaFunkcji (parametry) {

return treść

}



Aby wywołać funkcję podajemy po prostu jej nazwę i ewentualnie argument w nawiasach:
nazwaFunkcji()



Parametry podajemy podczas tworzenia funkcji.

Argumenty podajemy podczas wywołania funkcji.


return tak samo jak map() zwraca nam nową tablicę, z którą możemy pracować, return zwraca nam wartość, z któej możemy w przyszłości skorzystać. Return kończy działanie funkcji, wszystko pod return przestaje działać.
